# 系统架构设计

## 项目愿景

打造一个**数字 psyche 平台**，基于独创的 **MindWeave（思维编织）** 理论，用户可以：

1. 上传个人数据，通过六维思维线索提取，编织专属数字 psyche
2. 与名人 psyche 进行深度对话，体验不同思维方式的碰撞
3. 在 psyche 社会中测试言论的影响力和传播路径

---

## 核心概念：MindWeave 理论

### 思维编织（Mind Weaving）

每个人的思维就像一张独特的织物，由六维线索交织而成：

| 维度 | 说明 | 提取来源 |
|------|------|----------|
| **思维内核** | 核心认知模式、思考方式 | 深度访谈、文章 |
| **表达风格** | 语言习惯、修辞特点 | 聊天记录、社交媒体 |
| **决策逻辑** | 判断模式、选择偏好 | 决策记录、案例分析 |
| **知识图谱** | 知识体系、认知边界 | 专业文章、书籍 |
| **价值体系** | 价值观、原则、底线 | 公开表态、批评观点 |
| **情感模式** | 情感反应、情绪表达 | 聊天记录、访谈 |

### 数字 psyche（Digital Psyche）

通过思维编织技术构建的完整数字意识，包含：
- **思维内核（Mind Kernel）**：核心认知框架
- **表达引擎（Expression Engine）**：风格化语言生成
- **决策模块（Decision Module）**：判断和选择逻辑
- **知识网络（Knowledge Network）**：关联知识存储
- **价值守护（Value Guardian）**：伦理边界控制
- **情感模型（Emotional Model）**：情绪反应模拟

## MindWeave 引擎详解

### 六维线索提取流程

```
原始数据（聊天记录、文章、社交媒体）
    │
    ├──→ 线索提取引擎
    │       ├──→ 思维内核分析器
    │       │       └──→ 提取：思维框架、认知偏好、独特思考特征
    │       │
    │       ├──→ 表达风格分析器
    │       │       └──→ 提取：语气、句式、词汇、修辞、互动模式
    │       │
    │       ├──→ 决策逻辑分析器
    │       │       └──→ 提取：决策优先级、风险态度、时间偏好
    │       │
    │       ├──→ 知识图谱构建器
    │       │       └──→ 提取：专业领域、认知边界、知识组织方式
    │       │
    │       ├──→ 价值体系提取器
    │       │       └──→ 提取：核心价值观、道德底线、反对模式
    │       │
    │       └──→ 情感模式分析器
    │               └──→ 提取：情感基调、强度、触发点、表达方式
    │
    ├──→ 线索验证
    │       ├──→ 跨源一致性验证
    │       ├──→ 预测力验证
    │       └──→ 独特性验证
    │
    └──→ 思维编织
            ├──→ 构建思维内核（系统提示）
            ├──→ 配置表达引擎（风格参数）
            ├──→ 初始化决策模块（启发式规则）
            ├──→ 构建知识网络（向量库）
            ├──→ 设定价值守护（约束条件）
            ├──→ 配置情感模型（情绪参数）
            └──→ 生成 psyche 织物（可运行的数字意识）
```

## 数据库设计

### psyche 表

```sql
psyche (
    id: UUID PRIMARY KEY,
    user_id: UUID REFERENCES users(id),
    name: VARCHAR,
    psyche_type: ENUM('personal', 'celebrity', 'system'),
    status: ENUM('draft', 'weaving', 'ready', 'archived'),
    
    -- 思维织物
    mind_kernel: TEXT,              -- 思维内核（系统提示）
    expression_config: JSONB,       -- 表达引擎配置
    decision_config: JSONB,         -- 决策模块配置
    knowledge_network_id: VARCHAR,  -- 知识网络 ID
    value_guardian: JSONB,          -- 价值守护配置
    emotional_model: JSONB,         -- 情感模型配置
    
    -- 六维线索
    mind_threads: JSONB,            -- 存储六维思维线索
    
    -- 统计
    interaction_count: INTEGER DEFAULT 0,
    
    created_at: TIMESTAMP,
    updated_at: TIMESTAMP
);
```

## 核心功能流程

### 1. 编织个人 psyche

```
1. 用户上传数据文件
   └──→ 系统异步解析文件，提取文本内容
   
2. 思维编织流程启动
   ├──→ 六维线索并行提取
   ├──→ 思维内核分析
   ├──→ 表达风格分析
   ├──→ 决策逻辑分析
   ├──→ 知识图谱构建
   ├──→ 价值体系提取
   ├──→ 情感模式分析
   
3. 线索验证与校准
   
4. psyche 织物生成
   └──→ 用户可与自己的 psyche 对话测试
```
